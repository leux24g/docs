---
title: "Authentication"
description: "Generate a Waftpay access token."
---

You need a **Bearer access token** to call the Waftpay API.

## Steps
1. In the **Waftpay Dashboard**, create a **Client ID** and **Client Secret** (use Sandbox first).
2. Exchange them for a short-lived token:

```bash
curl -X POST https://sandbox.waftpay.io/api/oauth/token \
  -H "Content-Type: application/json" \
  -d '{"grant_type":"client_credentials","client_id":"YOUR_CLIENT_ID","client_secret":"YOUR_CLIENT_SECRET"}'
